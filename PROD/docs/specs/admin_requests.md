
# Admin Requests Specification

## Sub-Navigation (Tabs)
All'interno della sezione "REQUESTS", devono esserci 3 grandi tab di stato:
1. **TO DO**: Nuovi ticket aperti.
2. **IN PROGRESS**: Ticket presi in carico.
3. **DONE**: Ticket risolti.

## Request Detail View (Modal/Expand)
- Al click su una riga, si apre un dettaglio che mostra:
  - Messaggio completo dell'utente.
  - Dati utente (Email, ID).
  - Azioni: "Mark as In Progress", "Mark as Done", "Reply".
