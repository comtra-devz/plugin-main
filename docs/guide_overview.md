# Project Overview & Strategy

## Objective
1. **Audit**: Advanced scan (tokens, naming, a11y, duplicates).
2. **Refactor**: Auto-suggestions for improvements.
3. **Generation**: AI screens respecting DS tokens.
4. **Code**: React/Vue/Angular + Storybook sync.
5. **Learning**: Train AI on designer patterns.

## Dev Strategy
1. Start with Audit Engine + JSON Export.
2. Clean the DS before integrating AI.
3. Keep Code Gen optional initially.
4. Log everything for historical scoring.
