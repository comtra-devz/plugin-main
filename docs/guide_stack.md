# Tech Stack

## Frontend (Plugin)
- **Framework**: React (TypeScript).
- **Build**: Vite/Webpack.

## Backend (Antigravity)
- **Runtime**: Node.js / Cloud Run / Vercel.
- **DB**: Supabase (Postgres) or Firebase.
- **Audit**: `axe-core` + Custom JS logic.

## AI & Logic
- **LLMs**: OpenAI, Anthropic Claude, or Local (GPT4All).
- **Templating**: Handlebars.js / EJS (Code Gen).

## Infra & Costs
- **Figma API**: Free.
- **DB/Hosting**: Free tiers (Supabase/Vercel).
- **AI**: Usage-based.
